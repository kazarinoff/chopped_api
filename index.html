Episode Resource Parameters

Name Description Type
SeasonThe season it aired (duh). Seasons run numerically from 1 to 42 with additional "Specials" and 5 "Grill Masters" seasonsinteger, except for the specials and grill masters which are strings
SeriesnoThe order in the entire seriesinteger, except for the specials which are number sequentially on their own and prefaced w/ "S"
SeasonnoOrder in its seasoninteger
TitleEpisode's Titlestring
PremiereFirst air datedatetime
CoursesThe ingredients for each coursedictionary with keys for "Appetizer","Entree", and "Dessert", with values as arrays if ingredient strings
JudgesThe judges for that episodearray of name strings
ContestantsThe chefs in order of elimination (i.e. first out after appetizer, second out after entree, third out after dessert, last is winner)array of name strings
NotesAny applicable notes for the episodestring

Endpoints

zachkaz/com/chopped/api/{seriesno}one particular episode
zachkaz/com/chopped/api/?title={title}one particular episode (exact match only)
zachkaz/com/chopped/api/{season}all episodes of a particular season
zachkaz/com/chopped/api/indexall episodes (roughly 500)
zachkaz/com/chopped/api/?judge={name}all episodes judged by them (exact match only)
zachkaz/com/chopped/api/?contestant={name}all episodes cooked by them (exact match only)
zachkaz/com/chopped/api/?ingredient={name}all episodes featuring that ingredient (all matches)
